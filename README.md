# bash-tools
Random tools to make the cmd line more awesome. Clone this repo to your user home and add
```bash
source ~/bash-tools/bashrc.sh
```
to your `~/.bashrc` or `~/.bash_profile`

# misc notes ...

```bash
PATH="/usr/local/bin:$PATH" ; export PATH
```
